## Installination :

```
pkg install -y wget 
wget https://raw.githubusercontent.com/Learn-Termux/metasploit-termux/main/install-metasploit.sh
bash install-metasploit.sh
```
<br/>



## For Uninstall :

```
apt purge ruby
rm -rf $HOME/metasploit-framework
rm -rf $PREFIX/lib/ruby/
```
